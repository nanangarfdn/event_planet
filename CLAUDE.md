# Event Planet

## Commits

- One line only. No body, no footer, no co-author lines.
- Start with a conventional prefix: `feat`, `fix`, `docs`, `test`, `refactor`, `style`, `chore`, `perf`, `build`, `ci`.
- Format: `<prefix>: <short imperative summary>`.
- Examples:
  - `feat: add event detail screen`
  - `fix: correct pax limit for normal users`
  - `docs: update README`
  - `test: add negative cases for create wizard`
  - `chore: bump react-native-svg`
